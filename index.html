Git

Podstawy gita wystarczajace do prostej komunikacji z repo

Korzystamy z git bash, link do naszego repo to https://github.com/grrreg94/Front-Basic, Podstawowe komendy jakich na tym etapie uzywam to

git init, To polecenie zainicjuje lokalne repozytorium git

git status, To polecenie powie Ci jakie zmiany zostały już zapisane przy pomocy narzędzia git, a jakie jeszcze nie.

git add plik.typ, dodaje plik, Musze sie dowiedziec dlaczego za kazda zmiana pliku index git chce abym dodal ten plik!

git commit -m"Komentarz", zapisz zmiany

git remote add origin link_to_your_remote_repository, Ta komenda sprawi, że Twój lokalny git będzie wiedział gdzie znajduje się zdalne repozytorium

git push origin master, Komenda ta spowoduje, że wszystkie zapisane zmiany znajdujące się na lokalnej gałęzi master zostaną wysłane na gałąź master na zdalnym repozytorium o nazwie origin.

git pull origin master, To polecenie spowoduje, że wszystkie zmiany z gałęzi master ze zdalnego repozytorium o nazwie origin zostaną pobrane do Twojego lokalnego repozytorium na gałąź master.
